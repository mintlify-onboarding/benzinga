## Models
### BarsResponse:object

<ResponseField name="symbol" type="string">
The given symbol
</ResponseField>

<ResponseField name="interval" type="number">
The candle interval
</ResponseField>

<ResponseField name="candles" type="object[]">
An array of bar data / candles
</ResponseField>

<ResponseField name="candles[].open" type="number(float)">
Price at same day market open
</ResponseField>

<ResponseField name="candles[].high" type="number(float)">
Indicator of highest price of security for same day
</ResponseField>

<ResponseField name="candles[].low" type="number(float)">
Indicator of lowest price of security for same day
</ResponseField>

<ResponseField name="candles[].close" type="number(float)">
Price at same day close. Only available after market close.
</ResponseField>

<ResponseField name="candles[].volume" type="integer">
Volume for the current or previous regular-only trading session
</ResponseField>

<ResponseField name="candles[].time" type="integer">
Unix timestamp in millis.
</ResponseField>

<ResponseField name="candles[].dateTime" type="string (YYYY-MM-DDTHH:MM:SS.mmm-ZZ:zz)">
Datetime is aligned to the start or the candle, or to the start of the pre-market session if a daily.
</ResponseField>

<Accordion title="Structure (JSON)">
```json
{
  "symbol": "string",
  "interval": "number",
  "candles": [
    {
      "open": "number (float)",
      "high": "number (float)",
      "low": "number (float)",
      "close": "number (float)",
      "volume": "integer",
      "time": "integer",
      "dateTime": "string (YYYY-MM-DDTHH:MM:SS.mmm-ZZ:zz)"
    }
  ]
}
```
</Accordion>
